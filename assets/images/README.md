# Images

Place your images in this folder:
- `assets\images\IMG-20250628-WA0090.jpg` — Your profile photo (square, min 400×400px) shown in the hero section
- `about.jpg`   — A photo of you for the About section  
- `project1.jpg` — Screenshot of Project 1
- `project2.jpg` — Screenshot of Project 2
- `project3.jpg` — Screenshot of Project 3

The site will show placeholder graphics if these images are missing, so it still looks great!
